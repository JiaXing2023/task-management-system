# 📋 任务管理系统

一个基于 **Django + Vue3** 的企业级任务管理平台。

## ✨ 功能特性

### 👤 用户管理
- 用户注册/登录（账号密码 + 手机验证码）
- 个人资料修改（用户名、手机号、密码）
- 账号注销功能

### 👥 权限管理
- 管理员/普通员工双角色
- 角色权限控制（6种权限模块）

### 📝 任务管理
- 任务的增删改查
- 任务状态：待完成、已完成、已逾期
- 任务搜索筛选

### 📊 数据统计
- 任务状态分布饼图
- 月度任务完成趋势图
- 图表导出功能

### 📑 报表导出
- 支持 Excel/PDF/Word 格式
- 按任务类型和时间范围筛选

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| Django 4.2 | 后端框架 |
| Django REST Framework | API接口 |
| Simple JWT | JWT认证 |
| MySQL | 数据库 |
| Vue 3 | 前端框架 |
| ECharts | 数据图表 |
| Axios | HTTP请求 |

## 📦 安装部署

### 环境要求
- Python 3.9+
- MySQL 8.0+
